The amount of data that can be displayed with just a single image is impressive.
We can explore data on more than 1200 major films of the last ten years.
We are able to group films by genre, script type and oscar winners, assigning them unique colors for each genre script type, oscar winners. This makes grouping and discrimination between these categories clear and easy. Moreover its interactive as when you hover over an image, more info is displayed.

We also have the option to change X-Y axis values, and view more data in the same page.

User can select between % budget recovered, Worldwide Gross ($Million), Domestic Gross ($Million), Foreign Gross ($Million), Opening Weekend ($Million) Average Audience %, Budget million for Y axis data and between Average Critics %, Rotten tomatoes %, Metacritic %, Rotten tomatoes Vs Metacritic % Deviance, Audience Vs Critics Deviance %, Average Audience %, Budget ($Million) in the X axis.

There is also an option to filter results between 2008 and 2016.
In general the concept of X-Y axis to display the data is really powerful and efficient.
Most of us are familiar with these types of diagrams, as we go on the right side of X axis values are increased and the same applies to Y axis as we go up.

So we can easily identify the correlation between data in X and Y.
